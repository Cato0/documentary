WebPack

Start

npm run build

Standard Procedure

https://www.sitepoint.com/webpack-beginner-guide/

  1. npm init to create a project with an index.html and app.js inside of it

  2. install webpack

    npm install webpack webpack-cli --save-dev

  1. Add the following inside package.json -> scripts

    "dev": "webpack --mode development", "build": "webpack --mode production"

Afterwards it should look something like this now

{ "name": "learn-webpack", "version": "1.0.0", "description": "", "main": "index.js", "scripts": { "test": "echo "Error: no test specified" && exit 1", "dev": "webpack --mode development", "build": "webpack --mode production" }, "keywords": [], "author": "", "license": "ISC", "devDependencies": { "html-webpack-plugin": "^3.2.0", "webpack": "^4.41.2", "webpack-cli": "^3.3.9" } }

  1. npm run dev to start webpack in development mode

  1. npm install babel-loader @babel/core @babel/preset-env --save-dev

  2. inwebpack.config.js add the following after Output:

module: { rules: [ { test: /.js$/, exclude: /node_modules/, use: { loader: 'babel-loader', options: { presets: ['@babel/preset-env'] } } } ] },

  1. npm run dev -- --devtools false

  1. npm install css-loader style-loader --save-dev

  2. add the following to webpack.config.js inside module -> rules

    { test: /.css$/, use: [{ loader: 'style-loader' }, { loader: 'css-loader' }], },

  1. npm install file-loader --save-dev

  2. Add the following Rule to webpack.config.js

    { test: /.(png|jpg|gif)$/, use: [ { loader: 'file-loader' }, ], },

14. Dev 
npm install webpack-dev-server --save-dev

  1. package.json edit dev to:

    "dev": "webpack-dev-server --mode development",

  2. webpack.config.js add

    devServer: { contentBase: './dist', open: true },

  1. npm install --save-dev clean-webpack-plugin

  2. in webpack.config.js

    const CleanWebpackPlugin = require('clean-webpack-plugin');

    ...

    plugins: [ ... new CleanWebpackPlugin() ],

  1. npm run build
